Predicts the 2026 World Cup with an calibrated elo system made with a dixon coles poisson model fit on a data set of past international soccer games. Each team has a attack/defense strength with exponential time-weighting, and so recent form matters mroe than old results. Then, 20,000 indepedent simulations of all 12 groups with Fifa tiebreakers. 

Currently, it's in the first phase of group stage, but after this it will predict the rest of the tournament, updates percent with every game but it made a first prediciton/
